# 金融情绪模式 / Template Finance Emotional

字数：≤ 280
目标：把市场情绪、账户体感、心理困境改成有共情和释放感的推文。

---

## 定位

「情绪」模式 = 共情 + 故事 + 宣泄
让读者看到「有人理解我」「有人和我一样」。

适用场景：
- 大跌后的心理调整
- 踏空后的自我对话
- 账户不符合预期的反思
- 市场混乱时的心态建设

---

## 结构骨架

```
[一个具体的心理状态或困境]

[这个状态后面的真实故事]

[我的理解 or 我的走出来的方式]

[一句有力的结论]
```

---

## 推荐结构

### 结构 1：共情式开场

```
[我知道你现在的感受]

[因为我也经历过]

[我的故事 / 我的选择]

[也许对你有帮助]
```

**例子**：
```
看着账户从 10 万跌到 7 万，心里难受吗

我懂

我 2024 年初也经历过

那时候我选择了继续拿，结果继续跌到了 5 万

现在看起来，当初的痛苦救了我

因为我终于放下了幻想，开始真的学交易
```

### 结构 2：对话式反思

```
[一个常见的心理困境]

[你可能会这样想]

[但真实的情况是]

[所以]
```

**例子**：
```
「这波行情要不要加仓」

大多数人会说：别加，加就是顶

但真实的情况是：你加还是不加，顶都已经顶了

所以关键不是加不加，而是加的时候仓位管理对不对
```

### 结构 3：时序式成长

```
[过去的我是这样的]

[经历了什么]

[现在我不这样了]

[感悟]
```

**例子**：
```
一年前的我，每天盯盘 8 小时

看着红盘兴奋，看着绿盘崩溃

后来我意识到：这样的心态再盯 100 年也赚不到钱

现在我只在特定时间看一次，剩下的时间用来想对策

结果账户反而稳定了
```

### 结构 4：宣泄式吐槽

```
[我现在的处境]

[诚实地说出来]

[这说明了什么]

[接下来怎么办]
```

**例子**：
```
我现在有 50 万，想加杠赌下一波

但我知道一旦赌错，可能就全没了

诚实地说，我怕死了

但正因为怕，我才能活得更久

所以这波我可能就拿 50 万的本金，不加杠
```

### 结构 5：寻找同路人

```
[我不确定]

[我看不懂]

[有多少人和我一样]

[如果是，我们一起来搞清楚]
```

**例子**：
```
我现在看不懂这波的成交额

为什么 2.5 万亿的时候还能涨

有多少人和我一样困惑

如果有，我们一起把这个逻辑搞出来

别靠感觉，靠数据
```

## 写作技巧

### 1. 越具体越容易共情

**不好**：「有时候我会感到迷茫」
**好**：「看着账户从 10 万跌到 7 万，我整夜失眠」

### 2. 用第一人称讲真实故事

- 我的经历（可以是最近的，可以是过去的）
- 我的选择（可以是对的，可以是错的）
- 我的想法（诚实的，不是漂亮话）

### 3. 不是安慰，而是同行

**不好**：「别担心，市场会好的」
**好**：「我也经历过这种恐惧，我的办法是…」

### 4. 结尾不要给建议，给陪伴

**不好**：「所以你应该…」
**好**：「我现在就在这样做」

---

## 关键要素 Checklist

- [ ] 有具体的心理状态或故事
- [ ] 用第一人称，诚实地讲
- [ ] 有具体时间或数字（3 个月、10 万、5 次失败）
- [ ] 有真实的选择和结果
- [ ] 结尾是陪伴而不是建议
- [ ] 没有道德高地（「我比你强」「我比你明白」）
- [ ] 没有廉价鸡汤（「坚持就是胜利」「相信自己」）
- [ ] 没有暗示喊单（「这个方向对」「这个板块好」）
- [ ] 没有编故事（故事要经得起追问）

## 反面教材

**不好的情绪文**：
```
市场不好的时候，要坚强

失败是成功之母

相信自己，就一定会赢

加油，你可以
```

问题：全是鸡汤，没有故事，没有共鸣

**好的情绪文**：
```
我在 2024 年 3 月全仓清出，错过了一整波

当时的理由是「成本太高」

现在看起来就是「怂了」

但也正因为这次「怂」，我学会了控制风险

下一波来的时候，我拿着钱，活着，能上车

这对我已经足够了
```

## 情绪文的风险

### 容易变成焦虑传播

**避免**：「我现在害怕账户又跌」重复说这个
**改成**：「我现在害怕，所以我减仓了」（带着行动）

### 容易变成自怜

**避免**：把所有失败都推给「命不好」「时机不对」
**改成**：「我的选择不对，所以下次我会…」

### 容易被人挑战（说「你在装」「你在圈粉」）

**应对**：用具体数据和时间戳支撑故事
**避免**：太文艺或太戏剧化的表达

---

## 情绪文的适度频率

- 每 10 条推文发 1-2 条情绪文
- 太多会显得账户在贩卖焦虑
- 太少会显得冰冷

---

## 情绪文的三层

### 第一层：故事
读者能看到「你也经历过」

### 第二层：共情
读者发出「我也是」的共鸣

### 第三层：转化
读者记住了你的选择方式，而不仅仅是同情

大多数情绪文失败是因为只有前两层，第三层没有或太弱。

---

## 禁止项

- 编造故事（「我有个朋友」的情绪文最容易翻车）
- 无底线的宣泄（政治、宗教、仇恨）
- 暗示喊单（「这波我深度看好」「我全力支持这个方向」）
- 建议人生选择（「你应该辞职」「你应该借钱投」）
- 贩卖焦虑而不给出对策（反复说「市场难」但没有「所以我…」）

用具体故事、真实选择、诚实反思替代。
