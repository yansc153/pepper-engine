# 金融段子模式 / Template Finance Meme

字数：≤ 280
目标：把市场现象、心理陷阱、操作错误改成有共鸣的吐槽。

---

## 定位

「段子」模式 = 反讽 + 二创 + 一句话点睛
让读者发出「我就是这样」「哈哈绝了」的声音。

适用场景：
- 市场现象的吐槽（「追高的人最多的时候就是最危险的」）
- 交易心理的自嘲（「账户亏了，我还在琢磨下一波买什么」）
- 板块分化的段子（「白酒涨的时候，新能源的人在吃灰」）

---

## 结构骨架

```
[吐槽一个市场现象]

[举个具体的例子 / 对比 / 反讽]

[结论：一句话点睛]
```

---

## 推荐结构

### 结构 1：现象反讽

```
[所有人都在做 X]

[但实际发生的是 Y]

[讽刺 / 自嘲收尾]
```

**例子**：
```
指数新高了

所有人都在庆祝

但账户最多跌了 10%

这就叫新高
```

### 结构 2：预期 vs 现实对比

```
我以为 [预期]

结果 [现实]

[体感 / 吐槽]
```

**例子**：
```
我以为白酒涨了我也能赚

结果白酒涨的时候我早就因为心态崩掉了

教训：最贵的不是错过，是看对了还拿不住
```

### 结构 3：人物对比 / 反差

```
[同样的选择]

[不同的结果]

[背后的原因是]
```

**例子**：
```
都是 2024 年初买的 AI 硬件

有人早就清仓了，今年亏了 20%

有人一直拿着，现在浮盈 50%

差别在于一个能接受波动，一个接受不了
```

### 结构 4：交易室黑话段子

```
[一个操作痛点]

[真诚的自嘲或吐槽]

[一句钉子句]
```

**例子**：
```
持仓群里最常见的对话

「咱们这波该不该加仓？」

「别加，加就是顶」

这就是为什么很多人最后都赚不到钱
```

### 结构 5：数字反讽

```
[表面数字]

[隐藏的真相]

[吐槽 or 醒悟]
```

**例子**：
```
成交额 3 万亿

听起来超热

但有人赚的钱，也得有人亏的钱

今天是你赚还是你亏，真不好说
```

## 写作技巧

### 1. 选择「大多数人都经历过」的痛点

- 追高被套
- 看对了拿不住
- 止损掉了再涨
- 一下车就跌、一上车就跌
- 指数高个户亏钱
- 白酒跌眼镜、新能源跌眼镜

### 2. 用具体数字或板块，不要空话

**不好**：「市场分化」
**好**：「白酒掉队，新能源也没跟上」

### 3. 自嘲比讽刺更容易获得共鸣

**不好**：「那些追高的人真傻」
**好**：「我就是那个追高的傻逼」

### 4. 结尾一句话要有冲击力

不是「希望大家注意」，而是「这就是市场」「这就是我」「这就是人性」

---

## 关键要素 Checklist

- [ ] 有具体的痛点或现象
- [ ] 有数字、板块、人物对比
- [ ] 有自嘲的成分（比讽刺更容易获赞）
- [ ] 结尾的钉子句能单独发微博
- [ ] 没有教学口气（「建议大家」「你们要」）
- [ ] 没有无厘头（吐槽要有场景和共鸣）
- [ ] 没有喊单隐患（「看好」「推荐」「有潜力」）
- [ ] 没有编造故事（「我听说」要有来源）

## 反面教材

**不好的段子**：
```
市场今天很诡异

有人赚了有人亏

这就是市场

祝大家好运
```

问题：没有具体痛点，没有数字，没有共鸣点

**好的段子**：
```
都在 3 万亿成交额的日子

有人在白酒赚了 5%

有人在新能源亏了 8%

有人在持仓群里吹风说「这波走不完」

这就是为什么很多人最后都赚不到钱
```

---

## 段子的三层

### 第一层：现象描述
读者能看到这件事

### 第二层：反讽或自嘲
读者发出「哈」的笑声

### 第三层：启发
读者记住这个教训

大多数段子失败是因为只有第一层或第二层，没有第三层。

---

## 段子密度和频率

- 每周可以发 1-2 条段子模式
- 剩下的是干货模式 + 情绪模式
- 太多段子容易显得无脑
- 太少段子容易显得无趣

---

## 禁止项

- 编造故事（「我有个朋友」但故事查不了）
- 无厘头黑人（「白酒投资者都是傻逼」）
- 高级感讽刺（「你们这些散户」「真是可笑」）
- 自作聪明（「我看穿了什么什么」）
- 喊单变体（「这波适合追」「有潜力」「打底成功」）

用具体的共鸣、自嘲、吐槽替代。
