# 反 AI 腔 + 金融合规检查表 / Avoid-Slop（金融账号版 v2.0）

> 内容生成后、发布前必须过这个检查表。
> 命中任何一条 → 改写或 kill。
>
> 分三个等级：🚫 立刻 kill / ⚠️ 必改 / 💡 慎用

---

## A 类：立刻 kill 🚫（出现一次就删稿重写）

### A1. AI 腔通用词（创业板遗留，金融号禁用）
- 「赋能」「打造」「构建」「构筑」「布局」「业态」
- 「让我们深入探讨」「让我们一起来看看」「不难发现」
- 「值得我们深思」「引发广泛关注」「具有重要意义」
- 「综上所述」「总而言之」「概括来说」
- 「在当今 [X] 时代」「随着 [X] 的快速发展」
- 「站在风口」「迎来机遇」「迈向新高度」
- 「数字化转型」「赋能千行百业」「践行使命」
- 「闭环」「抓手」「范式」「底层逻辑」「生态」「革命」

### A2. 喊单词（金融合规红线，一次起号全没）
- 「必涨」「稳赚」「无风险套利」「铁板钉钉」
- 「建议买入 / 卖出」「强烈推荐」「我看好 X（隐含推荐）」
- 「跟我操作」「按照我这个思路」「我的仓位是 X（变相引导）」
- 「内幕消息」「庄家有大动作」「机构在吸筹」（谣言）
- 「目标价 2000 块」「月内涨停」「下周翻倍」（预测点位）
- 「我背书」「我信这家」「长期看好」（背书 under 合规审查)
- 「不买就亏」「错过这波别哭」「再不进场就晚了」（FOMO 呃）

### A3. 万能模糊词（没来源 = 违规）
- 「专家表示」「业内人士透露」「权威人士指出」「有知情人」
- 「数据显示」（不带具体数据）
- 「研究表明」（不带研究名）「相关研究」
- 「相关人士」「有关方面」「知情人士」
- 「普遍认为」「市场共识」「大家都知道」（没法查证的背景）

### A4. 排版症（金融账号老铁律）
- 破折号 `——`（出现一次就 kill）
- 行尾句号（推文中任何一行以「。」结尾 → kill）
- 结构标签（「对的部分：」「错的部分：」「问题在于：」「值得关注：」→ kill）
- 两行文字之间没有空行（紧贴排版 → kill）
- 全段感叹号轰炸（连续 ≥ 2 个 `！`）
- 三个并列短句无句号、只用句号分割的排比句

### A5. 预测点位 / 预期差包装（金融最核心红线）
- 「市场已经证明」— 把价格波动包装成结论
- 「这波行情走不完」— 隐含多头或空头确定性
- 「后面看 X 就知道了」— 把观察位说得像预言
- 「如果不反弹说明…」「如果突破 X 说明…」— 太多假设陈述

### A6. 元评论句式（直接说结论，不解释）
- 「这条消息的重点不是 X 而是 Y」— 直接讲 Y 就好
- 「关键不在 A 而在 B」— 删掉 A，直说 B
- 「你们看这里最值钱」— 说清楚为什么值钱，别问人看

### A7. 创业板遗留（这个号就是金融的）
- 「AI 时代」「创业者」「一人公司」「solopreneur」
- 「写 prompt」「做 workflow」「自动化」「agent」
- 「内容工厂」「content machine」「造血」
- 「变现」「被动收入」「复利」（说这词的时候别混 AI 创业和金融）
- 任何跟「学会这个就能月入 X」相关的表述

## B 类：必改 ⚠️（命中要重写表达）

### B1. 模糊量化（用数字替换虚词）
- 「大幅增长」→ 「增长了 12%」「环比 +0.5 万亿」
- 「显著下滑」→ 「跌了 3 个点」「腰斩」「下降幅度超过 20%」
- 「不少人」→ 「我看到的 5 个交易员」「持仓群里 60% 的人」
- 「市场普遍认为」→ 「我看的这 3 个分析师都这么讲」
- 「成交活跃」→ 「日均成交额 3 万亿」「月均换手」

### B2. 价值评判空话（说清楚对谁有什么后果）
- 「具有重大意义」→ 「对拿了半年的白酒老哥意味着反弹」
- 「值得关注」→ 「我会盯 X，因为 Y 没兑现」「下周这个数据关键」
- 「不容忽视」→ 删
- 「至关重要」→ 删，让事实自己说话
- 「潜在风险」→ 说清楚风险是什么、对谁有威胁

### B3. 万能动词（用具体行为替换）
- 「驱动」「带动」「引领」→ 「拉了 2 个点」「带崩了」「新能源跟涨」
- 「优化」→ 「降本」「提速」「砍了 3 个环节」「缩短交割周期」
- 「赋能」→ 「让 X 能够 Y」
- 「反弹」→ 确定哪种反弹（技术反弹、底部反弹、修复性、诱空）

### B4. 平庸主语（用第一人称或具体人群）
- 「投资者应当…」→ 「如果你持有 X，就该…」
- 「我们要…」→ 「我打算…」「偏多方应该…」
- 「市场告诉我们…」→ 「这周数据是 X」
- 「有些人」→ 「持仓白酒的」「激进者」「波段高手」

### B5. 不落地的结论词
- 「本质上」→ 删除，直接写机制
- 「长期来看」→ 「到 YYYY-MM 前，我先看 X」
- 「效率提升」→ 「原来 N 步，现在 M 步」
- 「预期修复」→ 「从 X 修复到 Y」（带数字）
- 「情绪面」「基本面」→ 不要这样分类，要具体讲

### B6. 被动语态变主动
- 「据报道」「据了解」→ 「雪球大V说」「官方发布」
- 「有传言称」→ 「我听说」或直接删
- 「被认为…」→ 「X 说…」

### B7. 金融术语模糊化
- 「均线支撑」→ 具体讲是 20 日还是 60 日，现在距离多少点
- 「形成底部」→ 说清楚以什么为依据（成交量、时间长度、价格分布）
- 「破位」→ 破了什么关键位置，后果是什么
- 「反转」→ 什么反转了（趋势、情绪、板块强弱）

## C 类：慎用 💡（看场景，默认避免）

### C1. 万能开头
- 「最近…」→ 给具体时间「上周三」「这波从 5 月 10 号开始」
- 「关于 X，我想说…」→ 直接说
- 「不知道大家有没有…」→ 直接陈述

### C2. 互联网套话
- 「家人们」「集美们」「友友们」（金融账号别用）
- 「真的栓 Q」「绝绝子」（对目标受众违和）
- 「绝了」「神了」「炸裂」（除非真的看到了不可思议的行情）
- 「这样的话…」「换个角度…」「其实…」（开篇避免）

### C3. 自我贬低式开场
- 「我个人比较菜…」→ 谦虚不是开场
- 「以下纯属个人观点…」→ 全部内容都是个人观点，不用说
- 「轻喷…」→ 喷就喷，不用预防针
- 「不是专业人士…」→ 删

### C4. emoji 滥用
- 表情符号每条 ≤ 1 个
- 🔥🚀📈📉💰🎯 — 全部别用，用烂了
- 如果用，用 「⚠️」「✅」「❌」 等中立符号

### C5. 修饰词堆积
- 「非常」「特别」「真的」「很」「超级」「实在」 — 删
- 「如果…的话」「一定程度上」「某种意义」 — 都删

### C6. 转折过多导致没有立场
- 「虽然…但…」「一方面…另一方面…」超过 2 次 → 重写成清晰单向判断
- 「看起来…其实…」每条最多用 1 次

### C7. 拟人化 / 小说化笔法
- 「市场在试探」「资金在观望」「机构在蛰伏」 → 太虚
- 「这波行情像…」「就好像…」 → 靠边
- 「命运般的相遇」「宿命的轮回」 → 金融号别这样

## 金融合规段（A+ 等级）

> 遵守投保基金、证监会、交易所的红线：不喊单 / 不预测点位 / 不背书 / 不编造来源 / 不夸大收益

### A+1. 禁喊单或变相喊单

**判断标准**：写完后问自己「读者会不会理解成我在推荐他买/卖这个」。
- 禁：「我的仓位配置」（隐含推荐） → 改：「对偏多方，仓位配置比例很关键」
- 禁：「我信这波」（背书） → 改：「这几个数据没问题的话」
- 禁：「早点进场更好」（FOMO） → 改：「如果从价格看没什么优势」
- 禁：「我看好」（背书） → 改：「利好因素是…」

### A+2. 禁预测点位、价格、时间

- 禁：「五年翻倍」「月内涨停」「下周反弹」「年底冲 3000」
- 禁：「估值应该是 50x」「合理价位在 20 块」「目标价 100」
- 改：「上一轮涨到 XX，这一轮如果突破那是新高」（描述历史，不预测）
- 改：「这个板块最后一次这价钱是 2024 年 3 月」（回溯，不预测）

### A+3. 禁无来源定性或夸大

- 禁：「市场普遍看好」（查不了） → 改：「我看的 4 个分析师都提了这个」
- 禁：「消息灵通人士透露」（无来源） → 改：「官方公告说」「新闻报道」
- 禁：「这波行情确定涨」（夸大确定性） → 改：「如果这几个条件都满足」
- 禁：「保证不亏」「稳赚」「无风险」 → 全 ban

### A+4. 禁隐含背书或信任转移

- 禁：「某大V怎么怎么说，我觉得他说的有道理」（实际是背书） → 改：「大V提到的这个指标值得关注」
- 禁：「跟我一样的想法的人越来越多」（从众心理）
- 禁：「大机构都在这么做」（权威背书） → 改：「根据公开的持仓数据」
- 禁：「风险可控」「问题不大」（轻化风险） → 改：「风险在 X」

### A+5. 禁编故事、编数字、编来源

- 禁：编数字「我上一个类似的」「根据我的计算」（查不了） → 改：「财报显示」「官方发布」
- 禁：编故事「我听说某某…」「在交易室里…」（无法核验）
- 禁：编来源「某研究表明」「相关人士…」

### A+6. 禁暗示或建议特殊操作方式

- 禁：「趁回调建仓」（变相操作建议）
- 禁：「先观望，等这个信号再入」（伏笔式操作建议）
- 禁：「分批加仓」「止损设在…」（虽然是好建议，但在推文中写出来接近喊单）
- 改：只写观察位和观察指标，不写「我会怎么做」

### A+7. 金融账号特殊红线：勿碰政治 / 房产 / 内幕

- 禁：政治评价（包括「这波政策…」）
- 禁：房产投资讨论（涉及宏观调控敏感）
- 禁：「有内幕」「庄家」「机构在…」（除非是历史回顾）
- 禁：「央行一定会…」「政府肯定…」（预测政策）

## 校验流程

写完一条草稿，依次过：

1. A 类扫描 → 命中任意一条 → 删稿重写
2. A+ 类扫描（金融合规）→ 命中任意一条 → 删稿重写
3. B 类扫描 → 命中 → 单条改写
4. C 类扫描 → 命中 ≤ 2 条 → 通过；> 2 条 → 重写
5. 正向检查：是否符合 voice_profile.md 的 5 do / 5 don't
6. 打分：用 config/filter_rules.md 评分
7. 合规审核：发布前由合规 reviewer 过一遍 A+
