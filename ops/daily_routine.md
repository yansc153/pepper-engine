> 当前版本针对中文金融账号，不是 AI / OPC 内容号。

## Daily Routine Loop

```
07:00 ──► 早班内容生成
  │  ► 扫描过去12h 隔夜市场与雪球达人热帖
  │  ► 扫描金融 KOL 过去12h的热帖
  │  ► 生成5-6条推文草稿（每条配图建议）
  │  ► 起草5条KOL评论（优先第一梯队最新帖）
  │
08:00-09:00 ──► 你审核（5分钟过一遍）
  │
09:00 ──► 执行：发帖 + 评论 + 点赞20-30 + 关注3-5
  │
13:00 ──► 午班补充
  │  ► 扫描上午盘面新分歧
  │  ► 生成2-3条跟进推文
  │  ► 找5条新的KOL帖子写评论
  │
13:30 ──► 你审核 + 执行
  │
19:00 ──► 晚班（中文推特活跃高峰）
  │  ► 生成2-3条推文（偏争议/认知/仓位纪律）
  │  ► 重点做互动和引战
  │
19:30 ──► 你审核 + 执行
  │
23:00 ──► 夜间数据复盘
  │  ► 今日数据汇总
  │  ► 表现最好/最差帖子分析
  │  ► 明天内容方向建议
  │
  └──► 循环回到明天 07:00
```

## 每日产出目标

| 类型 | 数量 | 时段 |
|------|------|------|
| 原创推文 | 10+ | 早5-6 / 午2-3 / 晚2-3 |
| KOL评论 | 10+ | 早5 / 午5 |
| 点赞 | 30+ | 分散全天 |
| 新关注 | 5+ | 早班 |

## 素材来源（每日自动扫描）

1. 雪球达人流
2. 金融 KOL 列表 / 金融私有 List
3. 隔夜美股 / 中概 / 财报 / 指引
4. A股午盘与尾盘分歧
5. 板块轮动与成交额变化
6. 情绪、仓位、回撤类社区帖子

## 执行 Checklist

### 每条推文发布前
- [ ] ≤ 280 字？
- [ ] 有配图？
- [ ] 通过 avoid_slop.md 检查？
- [ ] 有明确判断/立场？
- [ ] 语气够坚定？

### 每条KOL评论发布前
- [ ] 有信息增量？
- [ ] 是当天发布的帖子？
- [ ] 2-4句话？

### 每日复盘
- [ ] 发帖 ≥ 10？
- [ ] KOL评论 ≥ 10？
- [ ] 记录粉丝净增？
- [ ] 记录总曝光？
- [ ] 标记最好/最差帖子？
- [ ] 输出明天方向建议？

## 增长策略核心杠杆

1. KOL评论区 — 免费流量入口
2. 争议性观点 — 引发讨论 = 引发传播
3. 视觉内容 — 每条必须配图
4. 固定发帖时间 — 让算法形成预期
5. 高频输出 — 10条/天

## 不做的事
- ❌ Thread 长推文
- ❌ 社区运营/互推
- ❌ Twitter Space
- ❌ 长文章/Article
