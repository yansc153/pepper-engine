# Playwright UI 规则 — Twitter/X 页面元素

> 每次 Playwright 走过一个页面路径后，把按钮/输入框的选择器记录到这里。
> 后续操作优先用已知选择器，不再截图定位。
> 截图只在首次探索或页面改版时使用。

---

## 通用规则

1. 每次截图后立刻 `os.remove(screenshot_path)` 删除
2. 等待元素用 `page.wait_for_selector()` 不用 `time.sleep()`
3. 页面加载完成用 `page.wait_for_load_state('networkidle')`
4. 所有操作前先检查元素是否存在，不存在则回退截图模式
5. 选择器失效时记录到 MEMORY.md，等待人工更新
